Starting Week

Date written: 2020-10-10

This week was our introduction to the course, Computer Science for Games, with multiple modules being introduced with various assignments. Of these, the Game Design & Development Course is definitely the most interesting to me because of the application of knowledge it requires:

  • Learning how games work
  • Learning the theory behind 'fun'
  • Implementing these ideas in a way that creates fun

Overall, I'm also very interested with how the course is going. Additionally, I've been putting some time into learning Rust, a low-level language similar to C or C++, with an amazing compiler that enforces safe-practices. I think that having knowledge in a more complex language, when compared with Javascript and Python, where I'm more experienced, will be a big benefit going forward. Having the knowledge of how to write safe and fast code is a big positive in my view.
